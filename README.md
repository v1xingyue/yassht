## 简介:

使用 [https://godoc.org/golang.org/x/crypto/ssh](https://godoc.org/golang.org/x/crypto/ssh) 实现的一套新的ssh组合工具包。
包括ssh , sshd, sshrsa sshadmin , sshloginshell , 完成一套基于linux 的跳板机实现。
